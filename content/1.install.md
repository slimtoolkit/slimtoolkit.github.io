## Container Registry

```bash
docker pull dslim/slim
```

## Package Managers

::code-group

```bash [Homebrew]
brew install docker-slim
```

::

## Update

If you already have SlimToolkit installed use the **update** command to get the latest version.

```bash
slim update
```
